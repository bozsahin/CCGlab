CCGlab downloads and installs

CCGlab download/install: Automatic download and install script.

READ BELOW before you run the install script.

Download this wee script only, unzip it in a directory where you want CCGlab installed. It contains just one bash script. Run it like this:
$ ./ccglab-download-and-install
It downloads and installs the latest version in your working directory, and names the subfolder CCGlab.
SBCL and rlwrap go to their default directories as defined in their packages.

If you have apt-get or brew, sed, wget and bash, the script above will download and install ccglab code, lalrparser code, SBCL code and rlwrap code.

You can check whether you have them by doing which app at command line where app is one of the names above.

Below is more detailed information.

CCGlab needs: (1) a real linux/unix environment, or a linux virtual box which you can install for Windows. (2) Some Lisp code.

The install package above takes care of (2). For (1), see the bottom of the page.

To fully automate the download and install, you need a package manager to install a Lisp as well, either apt-get (for linuxes) or brew (for Mac OS). Make sure you have one of them if you want fully automated install. Use e.g. `which apt-get' command to find out whether you have apt-get.

Automatic install first downloads then installs CCGlab. If you can sudo on your machine, i.e. if you can be a superuser, it can install SBCL Lisp and rlwrap as well. If you don't have that permission, it just does CCGlab and lalrparser.lisp downloads and install, and leaves the Common Lisp and rlwrap to you. So you can bypass Lisp installation in the install script if you already have one. (In that case, the installer asks for your Lisp's path.)

NOTE: It won't install CCGlab if there is no local Lisp. Follow this path if you already have a local ANSI Common Lisp (note: GNU lisps, GCL and CLISP, are not ANSI!).

rlwrap is optional, and highly recommended. It gives you command recall, which is VERY convenient.

The easy installer will guide you through these options, and issue warnings where necessary.

Some free (and very good ) ANSI Common Lisp cites and other goodies:

  • Steel Bank Common Lisp
  • Clozure Common Lisp
  • rlwrap (command recaller; very nice for CCGlab/Lisp's toplevel interface)
  • brew (package manager for MACOS X)

I did not include Lispworks Personal Edition here although it's free and ANSI, because it does not allow loading code from init files, which is what CCGlab does.

CCGlab has been tested in both Lisp systems, and in MacOS, Ubuntu and virtual boxes. (Cygwin, anyone?)

Development was done almost fully in SBCL.

Some linux environments for Windows:

  • Virtual box
  • Cygwin

As long as it gives you bash, wget, unzip, sed, apt-get/brew, anything's fine.
